# NomNom Landing Page

Landing page for [NomNom](https://nomnom.cc) — UCSD's student-run food delivery platform.

## Overview

NomNom connects Triton2Go orders with student riders who deliver food across campus. Students order via Triton2Go, request delivery in NomNom, and riders bring it to dorms. Flat $2.5 fee. @ucsd.edu only.

## Stack

- Vanilla HTML, CSS, JS
- Mobile-first, responsive
- Domain: nomnom.cc

## Local development

Open `index.html` in a browser or use a simple static server:

```bash
# Python
python -m http.server 8080

# Node (npx)
npx serve .
```

## Deployment

Configure the `nomnom.cc` domain to serve the static files (index.html, styles.css, script.js). Works with Vercel, Netlify, Cloudflare Pages, or any static host.
